Please copy the contents of this folder into your htdocs folder and host the website via XAMPP.
When you host the website use the first index.html file to start the website.

To start please take the following steps:
	1. Open Xampp
	2. Start Apache and MySQL
	3. Import pet_supplies.sql within phpmyadmin.
		You should see a database pet_supplies with a table called users.
	4. The website should be ready to test with localhost.
		Navigate to index.html within this folder using localhost.
